# Tetris Game (Web)

A simple Tetris game built with HTML, CSS, and vanilla JavaScript.

## Features

- Classic Tetris gameplay
- Keyboard controls
- Lightweight, no build step

## Project Structure

- `index.html` - App shell
- `styles.css` - Styling
- `script.js` - Game logic

## Run Locally

Open `index.html` in a browser.

## Future Goal

I plan to migrate this project to a React Native mobile app in a future release.
